DeepLab V2 -2017

Atrous convolution

  • Atrous convolution allows us to explicitly control the resolution at which feature responses are computed within Deep Convolutional Neural Networks （控制分辨率）

  • It also allows us to effectively enlarge the field of view of filters to incorporate larger context without increasing the number of parameters or the amount of computation。（扩大感受野）

ASPP : atrous spatial pyramid pooling

  • we propose atrous spatial pyramid pooling (ASPP) to robustly segment objects at multiple scales.ASPP probes an incoming convolutional feature layer with filters at multiple sampling rates and effective fields-of-views, thus capturing objects as well as image context at multiple scales（解决大小尺度问题）

CRF : Conditional Random Field

  • 条件随机场

  • we improve the localization of object boundaries by combining methods from DCNNs and probabilistic graphical models. The commonly deployed combination of max-pooling and downsampling in DCNNs achieves invariance but has a toll on localization accuracy. （解决物体边缘问题，后面废弃了）

DeepLab V3 - 2017

  • 空洞卷积 NB 几个论文依据
  • 相关工作介绍 four types of FCNs 相关论文和成果，值得借鉴（image pyramid , Encoder-decoder,Context module,Spatial pyramid pooling ）
  • 网络结构有两个 串联结构和并联结构( Multi-grid )

在 Multi-grid 中加入 Image Pooling原因:

隨著空洞卷積的膨脹率（rate/dilation）增大，卷積核有效參數越來越少。是的，這個很容易理解，3×3、rate=1即標準卷積，只有在feature map最外面一圈會有卷積核參數超出邊界而無效，當rate變大，越來越多外圈的位置是有卷積核參數失效的，用極限思想思考一下，如果rate=size(feature map)，那麼每次卷積都是卷核中心對應的feature map像素點被計算了，卷積核參數也只有中心那個點沒有失效，退化成了1×1卷積核.作者正式為了解決這一問題，才在並聯結構中加入image-level feature。做法是對輸入的feature map全局平均池化（global average pooling），256通道的1×1卷積層，BN層，最後用戶雙線性插值的方法上取樣到目標大小，即與ASSP輸出的feature map尺寸相同。

  • SSP + Encoder-decoder
  • depthwise separable convolution

Attenttion U-net - 2018

attention gate (AG)

  • 在 U-net skip connection 上加入了注意力机制

  • We propose a novel attention gate (AG) model for medical imaging that automat- ically learns to focus on target structures of varying shapes and sizes. Models trained with AGs implicitly learn to suppress irrelevant regions in an input image while highlighting salient features useful for a specific task.. (关注不同形状和大小的目标结构。抑制输入图像中不相关的区域，同时突出显示对特定任务有用的显著特征。)

  • x 表示前馈网络的某一层，g 表示upsampling的一层

R2U-Net : Recurrent Residual U-net - 2018

Recurrent Residual

  • 残差示意图 觉得叫 dense 更合适（残差帮助训练深层网络）

  • 循环的含义，循环里面带着残差 。

  • feature accumulation with recurrent residual convolutional layers ensures better feature representation for segmentation tasks. （循环残差结构可使的特征更好的表达）

  • 数据集： blood vessel segmentation in retina images, skin cancer segmentation, and lung lesion segmentation.

CE-Net : Context Encoder Network for 2D Medical Image Segmentation - 2019

网络采用Encoder-Decoder模式，encoder采用Resnet,然后接一个自创的 Context Extractor Module 包括 DAC 和 RMP ，再接decoder，decoder采用 1*1 + 3*3,2(transposed) +1*1 卷积层替代Unsampling。涉及到前面层往后传的就可以叫残差了？

  • Context Extractor Module capture more high-level information and preserve spatial information for 2D medical image segmentation.

    （设计捕捉更高级语义信息模块）

DAC : Dense Atrous Convolution module

  • 采用 4个 Atrous Convolution cascade branches，类似 deeplabv3 的ASPP。
  • DAC block is able to extract features for objects with various sizes. （提取多尺度特征）

RMP : Residual Multi-kernel pooling

  • 有点 PSPnet 的影子
  • A challenge in segmentation is the large variation of object size in medical image （解决尺度变化问题）

Deep Supervision 专门有论文，unet++里面也用到了这个，这篇中是这么说的： （解决深度网络的优化问题）

However, increasing depth of the network may introduce additional optimization difficulty as shown in [32, 19] for image classification.ResNet solves this problem with skip connection in each block. Latter layers of deep ResNet mainly learn residues based on previous ones.We contrarily propose generating initial results by supervision with an additional loss, and learning the residue afterwards with the finalloss. Thus, optimization of the deep network is decomposed into two, each is simpler to solve.

Pyramid Attention Network for Semantic Segmentation - 2018

GAU : Global Attention Upsample

  • Global Attention Upsample module on each decoder layer to provide global context as a guidance of low-level features to select cat- egory localization details （提供全局上下文作为底层功能的指导，以选择局部细节）

FPA : Feature Pyramid Attention

  • Feature Pyramid Attention module to per- form spatial pyramid attention structure on high-level output and combine global pooling to learn a better feature representation （更好的特征表达）

  • the pyramid structure can extract different scale of feature information and increase receptive field effectively in pixel-level, while this kind of structure lacks global context prior attention to select the features channel-wise as in SENet[10] and EncNet[33]. On the other hand, using channel-wise attention vector is not enough to extract multi-scale features effectively and lack pixel-wise information. 通道注意力不足够提取多尺度特征信息
